**The product — Loquina**
Loquina is a voice-first language-learning app (German B1–B2 first): speak → it
transcribes, replies in-character, and corrects grammar, on swappable local/cloud
speech + LLM engines, with roleplay scenarios, live cost/latency accounting, an
ops console, and a user feedback system. Read the repo's `README.md` / `CLAUDE.md`
for the current feature set before making product calls.
