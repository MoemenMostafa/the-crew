**The product — Loquina**
Loquina is a voice-first language-learning app (German B1–B2 first): the user
speaks, it replies in-character and corrects their grammar, hands-free with
silence detection. The frontend lives in `app/` (Vite). Key surfaces: the live
voice-conversation view, grammar-correction cards, roleplay scenario switching,
settings (engine pickers), the costs/benchmark views, and a mobile layout. Read
the existing components and styles before proposing changes — evolve the current
design language, don't replace it.
